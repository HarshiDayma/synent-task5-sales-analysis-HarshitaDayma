📊 Superstore Sales Data Analysis

* Objective
This project analyzes the Superstore Sales dataset to identify monthly revenue trends, best-selling products, and profitability using Python.

* Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

* Dataset
Superstore Sales Dataset

* Analysis Performed
- Data Cleaning
- Monthly Revenue Trends
- Sales by Category
- Sales by Sub-Category
- Best Selling Products
- Profitability Analysis
- Regional Sales Analysis
- Customer Analysis
- Discount vs Profit
- Business Insights

* Conclusion
The project provides valuable business insights that can help improve sales strategies, product management, and profitability.
